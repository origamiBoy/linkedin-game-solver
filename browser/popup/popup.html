About

Contains solvers for each LinkedIn puzzle game. Solutions are found:

  • Algrithimcally (using logic and code)
  • Programatically (taking answers directly from the webpage)
  • AI (for the words problems)

Some puzzles also allow solutions to be stored, and then later inputted without the delay required in solving them. It is designed to allow users to save scores to LinkedIn.

Save scores to LinkedIn

  1. Log out of LinkedIn
  2. Go to valid puzzle page
  3. Solve puzzle
  4. Log in to save score

Notes:

If solving multiple puzzles, ensure that you log out between each one.

An openAI API key is needed for some functionality, but it is not nessessary get the solution for each puzzle.
