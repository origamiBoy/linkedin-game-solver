About

Can solve any LinkedIn puzzle game using solver algorithms or AI. Can save scores to LinkedIn account

Use on LinkedIn

Multiple steps are needed to have scores officially saved on your LinkedIn profile.

  1. Logged out of LinkedIn
  2. Navigate to a valid puzzle link
  3. Solve puzzle (may require multiple attempts and/or an API key)
  4. Input direct solution if needed
  5. Log in to save
  6. Repeat the process for each puzzle
